# 🌐 Pratham Rathod — Developer Portfolio

A clean, responsive **personal portfolio website** built with pure HTML, CSS, 
and JavaScript — showcasing my skills, projects, and experience as a 
Java Backend Developer.

---

## ✨ Features

- 💼 Projects showcase with descriptions and links
- 🧑‍💻 About Me section with skills overview
- 📄 Resume download (PDF) directly from the site
- 📱 Fully responsive — works on mobile, tablet & desktop
- ⚡ Fast & lightweight — no frameworks, pure HTML/CSS/JS
- 🎨 Clean modern UI design

---

## 🚀 How to Run

**Option 1 — Open directly:**
1. Clone the repo:
```
git clone https://github.com/mrprathm/Pratham-Portfolio.git
```
2. Open `index.html` in any browser ✅

**Option 2 — Live Server (VS Code):**
1. Install **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
